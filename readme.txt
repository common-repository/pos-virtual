== Description ==
Integra tu POS Virtual a WooCommerce con este plugin desarrollado por ShopShop para aceptar pagos con tarjeta de crédito o débito en contado y cuotas.


== Installation ==


1. Have WooCommerce installed.
2. install this plugin, or add it by pressing the button that says, "Upload plugin".
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the Settings->"POS Virtual" screen to configure the plugin
5. When buying the Plugin, they sent a password to the registered purchase email.
6. Agree your credentials in the fields intended for each of them.
7. Click on "Probar Conexión".
8. If you want to update credentials, click on disable credentials and perform the process again.

== Description of services used ==

Se hace uso del [servicio](https://h.online-metrix.net/) en el archivo POSVirual/PosVirtual.php: lineas 910 y 916  para cumplir con la función de firmar con una huella digital puede mejorar la conversión de las ventas mediante un servicio especializado en la prevención del fraude. Este servicio lo provee CyberSource, el cual mitigará y gestionará los riesgos relacionados con el fraude On-line de una forma sencilla y práctica, utiliza la huella digital del dispositivo desde donde se está ejecutando la transacción. Es mandatorio enviarlo en cada transacción.

Url de este servicio: [Cybersource](https://support.cybersource.com/knowledgebase/knowledgearticle/?code=000003501)

Detalle del servicio [admlink](https://adm.posvirtual.gt/):
Integra nuestra solución para vender tus productos en línea por medio de nuestra pasarela de pagos. Estamos respaldado por Visanet para procesar transacciones en Epay Server o Cybersourse.


### Características
* Cobra de manera fácil y rápida dentro de tu sitio web.
* Permite que tus clientes completen el pago sin salir de tú sitio web.
* Permite llevar el control de tus ventas con reportes y estadísticas en tiempo real desde la administración de pos virtul
* Las transacciones son inmediatas y acepta tarjetas de diferentes bancos.
